public interface

CMISFacade

org.mule.modules.cmis.runtime.CMISFacade
Known Indirect Subclasses
CMISConnector, CMISConnectorCapabilitiesAdapter, CMISConnectorConnectionManagementAdapter, CMISConnectorLifecycleInjectionAdapter, CMISConnectorMetadataAdapter, CMISConnectorProcessAdapter, ChemistryCMISFacade
CMISConnector CMIS (Content Management Interoperability Services) is a standard for improving interoperability between ECM systems. 
CMISConnectorCapabilitiesAdapter A CMISConnectorCapabilitiesAdapter is a wrapper around CMISConnector that implements org.mule.api.Capabilities interface. 
CMISConnectorConnectionManagementAdapter A CMISConnectorConnectionManagementAdapter is a wrapper around CMISConnector that implements org.mule.devkit.dynamic.api.helper.Connection interface. 
CMISConnectorLifecycleInjectionAdapter A CMISConnectorLifecycleInjectionAdapter is a wrapper around CMISConnector that adds lifecycle methods to the pojo. 
CMISConnectorMetadataAdapter A CMISConnectorMetadataAdapter is a wrapper around CMISConnector that adds support for querying metadata about the extension. 
CMISConnectorProcessAdapter A CMISConnectorProcessAdapter is a wrapper around CMISConnector that enables custom processing strategies. 
ChemistryCMISFacade Implementation of CMISFacade that use Apache Chemistry Project.

public abstract Object folder (Folder folder, String folderId, NavigationOptions get, Integer depth, String filter, String orderBy)

Navigates the folder structure.

Parameters
folder Folder Object. Can be null if "folderId" is set.
folderId Folder Object id. Can be null if "folder" is set.
get NavigationOptions that specifies whether to get the parent folder, the list of immediate children or the whole descendants tree
depth If "get" value is DESCENDANTS, represents the depth of the descendants tree
filter Comma-separated list of properties to filter (only for CHILDREN or DESCENDANTS navigation)
orderBy Comma-separated list of query names and the ascending modifier "ASC" or the descending modifier "DESC" for each query name (only for CHILDREN or DESCENDANTS navigation)
Returns
  • the following, depending on the value of "get" parameter:
    • PARENT: returns the parent Folder
    • CHILDREN: returns a CmisObject ItemIterable with objects contained in the current folder
    • DESCENDANTS: List> representing the whole descentants tree of the current folder
    • TREE: List> representing the directory structure under the current folder.
